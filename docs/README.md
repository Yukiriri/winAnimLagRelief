从NVIDIA RTX30+开始，在低负载时存在系统动画掉帧  
掉帧的原因是GPU核心和显存都休眠太深，重新唤醒又过慢  
使用NV自带的工具锁定GPU核心和显存节能频率下限，可以大幅缓解掉帧  

## 用途一栏
[Lock-405&810]: (../bin/lock-nvgpu-405&810.bat)
[Lock-600&5000]: (../bin/lock-nvgpu-600&5000.bat)

| 基准参考    | 选择            | 待机功耗影响 |
| :---------- | :-------------- | :----------- |
| 1080p@300Hz | [Lock-405&810]  | 几乎未提高   |
| 2160p@160Hz | [Lock-600&5000] | 明显提高     |

> [!IMPORTANT]  
> 需要管理员权限运行  

> [!NOTE]  
> 这个修改仅够维持本次开机，严格来说只能维持本次驱动会话，驱动状态更改后就被重置  
> 如果要永久生效，需要加入开机自启
